OpenXM (Open message eXchange protocol for Mathematics): http://www.openxm.org

Welcome to the home page of OpenXM. Last update is September 11, 2000 JST.

What is OpenXM? It is an infrastructure for mathematical communication. (it is not XML, but it can be described by XML.) It is an experiment to connect mathematical software systems each other on the infrastructure. We are providing packages of mathematical software systems supporting the OpenXM protocol. Here is a quick tour of the OpenXM package: functions 1, functions 2, Efficient computation.
