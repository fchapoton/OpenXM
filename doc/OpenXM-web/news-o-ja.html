  • Debian パッケージ openxm-1.3.1-4 (Debian, 32bit, ネットワークインストーラ) は こちらから 入手できます. こちらに 注意事項が書いてありますのでご覧下さい. (Feb 23, 2010. openxm-1.3.1-4)
    D-加群についての機能が充実しました.
    1.3.1-3 以前がインストールしてある場合, 下記のようにインストールして下さい. 
         sudo su
     apt-get install make nkf
     dpkg --purge openxm
     apt-get install openxm
    TeXmacs インタフェースをインストールするには 
        apt-get install openxm-skel
    です.
  • Debian パッケージ openxm-1.3.1-1 (Debian, 32bit) は こちらから 入手できます. こちらに 注意事項が書いてありますのでご覧下さい. (Feb 12, 2010. openxm-1.3.1-1)
    D-加群についての機能が充実しました.
    apt-get update が使えないのでご注意. 必ず下記のようにインストールして下さい. 
         apt-get install make nkf
     dpkg --purge openxm
     apt-get install openxm
    TeXmacs インタフェースのユーザディレクトリへのコピーは 
        cp -r /root/.TeXmacs  ~/.
    です.
  • Debian パッケージ openxm-1.2.3-12 (Debian Etch/Lenny 用)は こちらから 入手できます. こちらに 注意事項が書いてありますのでご覧下さい. (Feb 24, 2009. openxm-1.2.3-12)
    種々の bug fix や改良の他に, 実験的機能として, 3D graphics, 加群のGB, D加群の積分などが追加されています. fres(resultant), taji_alc(algebraic local cohomology) などのパッケージとマニュアルが加わりました. 詳しくは helph() 関数でマニュアルをご覧下さい.
    apt-get update が使えないのでご注意. 必ず下記のようにインストールして下さい. 
         dpkg --purge openxm
     apt-get install openxm
    TeXmacs インタフェースのユーザディレクトリへのコピーは 
        cp -r /root/.TeXmacs  ~/.
    です.
  • Debian パッケージ opexm (Debian Etch 用)は こちらから 入手できます. こちらに 注意事項が書いてありますのでご覧下さい. (apt-get update が使えないのでご注意) (Jan 30, 2008. openxm-1.2.3-10)
    問題点: (1) gcc がインストールしてない場合(etch の標準デスクトップ環境など) "apt-get install gcc" で gcc をインストールして下さい.
  • asir-contrib の新しいパッケージ: taji_alc (algebraic local cohomology in 1 variables), f_res (resultants). ox_grep("taji_alc") or ox_grep("f_res"); で利用できる関数が見れます. (Jan 30, 2008)
  • Debian パッケージ opexm (Debian Etch 用)は こちらから 入手できます. こちらに 注意事項が書いてありますのでご覧下さい. (apt-get update が使えないのでご注意) (Aug 28, 2007)
  • VMware/knoppix/math 2006 を公開しました (March 24, 2006). VMware/knoppix/math は knoppix/math を VMwareの仮想HDへインストールし, Windows との連携のために, サンバサーバ等を加えた VMware の仮想HDイメージです.
  • cfep 1.1.2 を公開 しました(March 23, 2006). cfep は MacOS X 10.4 用の OpenXM ユーザインタフェースです. デフォルトの計算エンジンはRisa/Asir です.
  • Asir-contrib に y_prime (素数判定等), fj_curve (特異点解消の dual graph など), nk_mora (tangent cone algorithm for D, 局所b関数) が加わりました (Nov 11, 2005).
