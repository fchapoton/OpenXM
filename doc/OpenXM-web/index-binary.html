OpenXM Binary Distribution (tar+gzip format)

You need both OpenXM-shared.tgz and OpenXM-binary-(YOUR OS)-(YOUR-CPU).tgz to run the OpenXM package.

Installation Guide

    Architecture Independent Files (RELEASE_1_1_2) OpenXM-shared.tgz

  • Library Files, Java Class Files, Documents [ Library Files etc. (Shift-Click)].

    Binaries (RELEASE_1_1_2) OpenXM-binary-*.tgz

  • FreeBSD 3.3, i386. [FreeBSD 3.3 (Shift-Click)]. This binary is generated on FreeBSD 3.3-RELEASE. This binary does not contain math2ox and ox_math (binary files to support Mathematica).
  • Linux (libc 6), i386. [ Linux (glibc 2) (Shift-Click) ]. There might be troubles on shared libraries. Please report. This binary contains math2ox and ox_math (binary files to support Mathematica).
  • Solaris 2.5.1 SPARC [ Solaris (Shift-Click) ]. This binary contains math2ox and ox_math (binary files to support Mathematica). It does not contain an executable for PHC pack nor gnuplot. There might be troubles on shared libraries. Please report.
