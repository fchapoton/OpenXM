OpenXM Binary Distribution (tar+gzip format)

Windows 2000/XP

An experimental binaries of OpenXM for Windows 2000/XP, which include OpenXM/Risa/Asir. They support onlyl a subset of OpenXM. The full support is available only on the unix version.
  1. [ English version, OpenXM for windows (Shift-Click)].
  2. [ Japanese version, OpenXM for windows (Shift-Click)].

How to install OpenXM-win

Use winzip to extract the files from OpenXM-win-en.tgz (or OpenXM-win-ja.tgz). The extracted folder OpenXM-win should be put under c:\ See the readme-en.html in the folder OpenXM-win for further instructions. Do not copy OpenXM-win to the desktop or folders which use non-ascii characters and the space letter as their names. It is a developing version and will not work. If you do not have winzip and do not want to use it, an alternative is to get "tar.exe" and "gzip.exe" for windows and extract OpenXM-win by
tar xzvf OpenXM-win-en.tgz
in the command prompt. The files "tar.exe" and "gzip.exe" are obtainable from here

Unix

We recommend to use the source distribution on unix computers to avoid troubles on shared libraries. Unix binaries have not yet been ready for 1.2.2.
You need both OpenXM-shared.tgz and OpenXM-binary-(YOUR OS)-(YOUR-CPU).tgz to run the OpenXM package.
