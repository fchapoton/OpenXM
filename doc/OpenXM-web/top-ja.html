    1.3.1-3 以前がインストールしてある場合, 下記のようにインストールして下さい. 
         sudo su
     apt-get install make nkf
     dpkg --purge openxm
     apt-get install openxm
    TeXmacs インタフェース, desktop アイコンをインストールするには 
        apt-get install openxm-skel
    apt-get install openxm-gnome-desktop
    です.
    apt-get を使えない時(つかってはいけないとき)は, まず openxm- ではじまるすべてのファイルを http://fe.math.kobe-u.ac.jp/KnoppixMath からダウンロードしたあと, 次のようにして手動でインストールします. 
       sudo su
   dpkg --install openxm_*.deb
   dpkg --install openxm-skel*.deb
    なお openxm_*.deb はネットワークインストーラで本体は internet からダウンロードされます.

OpenXM とは?

数学に関する通信をやるための基礎技術です. 数学に関する通信をするためのプロトコルを提案しています. (OpenXM-RFC's). 数学データの標準化プロジェクトである OpenMath も応援してます. OpenXM では実際にいろいろな数学ソフトウエアをつないでパッケージを作成する 実験をやり, そのパッケージを配布しています. パッケージの画面コピーは こちら をご覧下さい. 以下はOpenXM パッケージの紹介のページです. functions 1, functions 2, Efficient computation.
