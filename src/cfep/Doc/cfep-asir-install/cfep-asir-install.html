cfep/asirのインストールの仕方

下の画像の ページ に行き、ダウンロードを開始するため, “cfepのアーカイブ”をクリックしてください。

すると、ダウンロードがはじまるので、それが完了したら左の画像の上の方のアイコンをダブルクリックすると(バージョンの数字部分は違う数字かもしれない)、 下のようなフォルダ(cfep-intel)が生成される.

フォルダを開くと、下の画像が現れるので、cfepのアイコンをディスクトップにドラッグしてコピーしてください。このとき、sampleもコピーしておくとすぐにcfepの動作確認ができるので便利でしょう。
