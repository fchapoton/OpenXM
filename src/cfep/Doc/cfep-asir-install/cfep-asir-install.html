cfep/asirのインストールの仕方

cfep/asirのインストールの前に、XcodeとX11をインストールする必要がある。

Xcodeのインストールの方法はこちら

X11のインストールの方法はこちら

下の画像のURL ここ に行き、ダウンロードを開始するため, “cfepのディスクイメージ”をクリックしてください。

すると、ダウンロードがはじまるので、それが完了したら左の画像の上の方のアイコンをダブルクリックすると(20061112 の部分は違う数字かもしれない)、 下のようなアイコン(cfep)が出てくる

出てきたアイコンを開くと、下の画像が現れるので、cfepのアイコンをディスクトップにドラッグしてコピーしてください。このとき、sampleもコピーしておくとすぐにcfepの動作確認ができるので便利でしょう。
