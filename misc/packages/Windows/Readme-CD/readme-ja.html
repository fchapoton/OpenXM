動作環境

Windows 版 OpenXM パッケージ は Windows 2000/XP で動作します. Windows 95/98/ME でも動作はしますが, コンポーネント間の通信速度が Windows 95/98/ME の制約により極端に遅いので, 動作が非常に緩慢となります.

CDROM での OpenXM 版 asir の起動方法.

Windows 版 OpenXM は ハードディスクにインストールすることなく CD のみでも利用できます.
  1. デスクトップのマイコンピュータアイコンをダブルクリックして 開き, その中の CDROM を開く.
  2. OpenXM-win フォルダ, 次に asir フォルダ, その次に bin フォルダ を開き, asirgui-cd をダブルクリックします. すると asir が起動します.

Asir のプログラムは メモ帳 (notepad) などのテキストエディタで書きます. CD の中のフォルダ \OpenXM-Archive には Micro-Emacs フォルダがあり, この中には mewin (Windows 版 microEmacs エディタ) が入っています. このテキストエディタを利用することも可能です.

OpenXM-win\bin-windows の下のバッチファイルは開発中の版のため原則動作しません.

OpenXM-win\asir\bin の下の asirgui は, Windows 版の最新の asir です.

Windows 95/98/ME で "環境変数領域が足りない" といわれたとき

  1. スタートメニュー/プログラム/アクセサリ より, "MSDOS プロンプト" を実行
  2. CDROM が D ドライブなら,
    command /e:6000
    d:
    cd \OpenXM-win\asir\bin
    を MSDOS プロンプトへ入力してから,
    asirgui-cd
    を入力すると asir が起動します.

ハードディスクへのインストール方法

  1. デスクトップのマイコンピュータアイコンをダブルクリックして 開き, その中の CDROM を開く.
  2. デスクトップのマイコンピュータアイコンをダブルクリックして 二つ目の ウインドウを開く.
  3. CDROM の OpenXM-win フォルダを, もう一つのウインドウの ローカルディスク (c:) へドラッグアンドドロップして, コピーします.
  4. ローカルディスク (c:) を開くと, OpenXM-win フォルダが出来ているので, その中の asir フォルダ, bin フォルダを順番に開き, asirgui アイコンをダブルクリックして asirgui を一度起動する. (これで registry に必要な情報が書き込まれてインストール終了. 失敗したら, regedit で HKEY_LOCAL_MACHINE->SOFTWARE->FUJITSU のなかの ASIR を全て消す.)
開発中版のため, デスクトップなどにはコピーしても動きません.

CDROM での OpenXM 版 kan/sm1 の起動方法.

  1. デスクトップのマイコンピュータアイコンをダブルクリックして 開き, その中の CDROM を開く.
  2. OpenXM-win フォルダ, 次に bin フォルダ を開き, sm1 をダブルクリックします. すると kan/sm1 が起動します.
