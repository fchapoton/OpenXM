この CD の使い方

フォルダ asir-book, フォルダ asir, フォルダ bin を順番に開くと, asirgui がある. これをダブルクリックすれば, asirgui が起動する. プログラムは メモ帳 (notepad) で書く. asirgui をハードディスクにコピーするには, asir-book のフォルダをそのまま, CD ドライブから ドラッグ アンド ドロップ で C ドライブ へコピーすればよい. 日本語を含むフォルダにコピーすると正しく動作しないので, デスクトップへはコピーできない デスクトップへショートカットを作成するのは, OK.

Meadow (emacs) のインストール方法

Meadow-1.14-i386-base.tar.gz は tar.gz という形式のアーカイブである. アーカイブというのは沢山のファイルやフォルダをまとめて一つのファイルに したものである. ここから, 必要なファイルをとりだすには, winzip を用いる. 展開先(とりだし先) は必ず, C:\ とするように. winzip を入手できない場合は, Meadow フォルダ(ディレクトリ)にある, tar.exe と gzip.exe を用いても展開できる. まず, tar.exe, gzip.exe, Meadow-1.14-i386.tar.gz を C:\ (C ドライブのルート) へコピーする. それから コマンドプロンプト を開いて, つぎのように入力すればよい. 

    cd c:\
    tar xzvf Meadow-1.14-i386.tar.gz
あとは, 新しくできあがった Meadow フォルダの INSTALL.Meadow の解説にしたがって インストールすればよい. (ようするに Meadow\1.14\install を実行すればよいということが書いてある.)
なお Meadow フォルダが存在しない, オンライン版の asir-book フォルダの場合は, Meadow-1.14-i386.tar.gz などの必要なファイルは, ここ からダウンロードできる. また google などから検索すれば最新版が入手できるであろう.
