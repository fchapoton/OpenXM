The OpenXM package is a package of mathematical softwares.
The OpenXM protocol provides a infrastructure for free, or open source
mathematical softwares.

As to installation procedures,
   see doc/install-eg.txt  in English
and
   see doc/install-jp.txt  in Japanese.

These files will be generated from install.oxweave by 
   cd doc
   make
in a source distribution.

As to Copyright notice, see the directory Copyright.
